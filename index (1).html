О системе ARPS
Advanced Return Prediction System

Что делает система

ARPS — интеллектуальная система прогнозирования возвратов для платформы электронной коммерции Uzum Market. При каждом подтверждённом заказе система за 80–120 мс анализирует 10 признаков покупателя и товара, вычисляет индекс риска возврата (0–100) и при значении ≥ 42 автоматически инициирует персонализированное вмешательство.

Разработана в рамках дипломной работы на данных 87 340 заказов Uzum Market за 2024 год.

Архитектура системы

1
Сбор данных
Транзакции из OMS, поведение из веб-аналитики, данные о товарах из каталога → Redis-кэш + ClickHouse
2
Feature Engineering
Batch-режим (агрегаты за 6 мес., ежедневно) + online-режим (признаки сессии, в реальном времени)
3
Модель XGBoost
ONNX-сериализация, REST API на FastAPI, инференс 80–120 мс, горизонтальное масштабирование
4
SHAP-объяснения
Топ-3 фактора риска для каждого заказа → персонализированное сообщение покупателю
5
Мониторинг
PSI-мониторинг дрейфа данных, ежеквартальный ретрейн модели, A/B-тестирование интервенций
